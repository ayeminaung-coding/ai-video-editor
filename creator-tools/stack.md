# Creator Tool Stack

High-level list of the tools you use as a faceless content creator.

## 1. Ideas & Scripts
- AI writing (ChatGPT / Claude / Gemini) — movie recap + meme scripts
- Google Trends, YouTube Shorts tab, TikTok search — topics & hooks

## 2. Voice & Audio
- Gemini Text-to-Speech — English + Burmese TTS
- (Optional) Audacity — clean or tweak recorded audio

## 3. Visual Assets
- Pexels, Pixabay, Coverr — stock / b-roll / cat clips
- Unsplash, Freepik (free items) — images & overlays
- OBS Studio / phone screen recorder — screen capture

## 4. Editing & Subtitles
- AI Video Editor (this web app) — trim, speed, filters, text, export
- CapCut / VN — backup mobile editor
- Auto-caption tools (CapCut, Whisper-based) — subtitles (EN + MM)

## 5. Thumbnails & Cards
- Canva — thumbnails, covers, title cards

## 6. Scheduling & Analytics
- YouTube Studio, TikTok app, Facebook Creator Studio — upload & schedule
- YouTube/TikTok analytics + TubeBuddy/VidIQ (free tier) — optimization

This file is a quick overview. Detailed workflows live in the other files in this folder.
